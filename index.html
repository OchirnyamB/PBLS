Welcome to Web Application Development

Web Application Development is part of the Governor's Career and Technical Academy of Arlington. The course consists of ITD 110 - Web Page Design I, ITP 100 - Software Design, and MTH 181 - Finite Mathematics. The purpose of the course is to provide students with a hands-on introduction to web page design, computer programming, and finite mathematics. Students are taught HTML, CSS, and Python through the use of a Command Line Interface.
